# Quick Start

## What do you need?

- python3
- python lib `requests`

```
pip install requests
```

- Add a new file `config.json` in `/src` and add your environments here:

```
{
  "mail_host": "smtp.***.com",
  "mail_user": "***@***.com",
  "mail_password": "***",
  "receivers": [
    "***@***.com",
    "***@***.com"
  ]
}
```
